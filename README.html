ğŸ”§ Troubleshooting

Socket.io Import Error

Use named import in backend/server.js:

import { Server as SocketServer } from 'socket.io'

PostCSS Plugin Error

Change plugin key in frontend/postcss.config.js:

// Change from:
'tailwindcss/postcss'

// To:
'tailwindcss'

Tailwind @apply Not Working

Ensure directives are in frontend/src/index.css, then:

# Clear Vite cache
rm -rf node_modules/.vite
npm install
npm run dev

MongoDB Connection Issues

Verify backend/config/database.js and MONGO_URI in your .env file.
